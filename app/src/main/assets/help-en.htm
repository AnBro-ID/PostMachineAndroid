Post machine emulator

Classic Post machine

The Post machine is an abstract computing machine based on the ideas of American mathematician E. L. Post. According to Post's thesis, any algorithm can be written as a program for the Post machine.

The Post machine consists of a reading/writing pointer and an infinite tape divided into cells. Each tape cell can either be empty (0) or contain a label (1).

Program for machine consists of numbered rows. Each row contains one of the following commands:

  • > N - move pointer to the right by 1 cell and go to the row N;
  • < N - move pointer to the left by 1 cell and go to the row N;
  • 0 N - write to the current cell "0" (erase label) and go to the row N;
  • 1 N - write to the current cell "1" (put label) and go to the row N;
  • ? N, M - if current cell contains "0", then go to the row N, otherwise go to the row M;
  • . - stop program execution.

You can omit the jump row number in the >, <, 0 and 1 commands. In that case next row will be executed.

Triple Post machine

Triple Post machine uses an extended alphabet consisting of three characters: space, "0" and "1". This allows you to program tasks in which numbers are written in binary notation. Triple mode additional commands:

  • X N - write a space to the current cell (erase label) and go to the row N;
  • 0 N - write to the current cell "0" and go to the row N;
  • 1 N - write to the current cell "1" and go to the row N.

if...else command contains three comma-separated row's numbers:

  • ? N, M, L - if current cell is empty then go to row N, otherwise if current cell contains a "0" go to row M, otherwise go to row L.

How to work with app?

In action bar menu there is an item "Task description", it contains editor field where you can enter task description in a free form.

Tape moves to the left/right using gestures (swipes). You can quickly go to the first encountered cell with "0" or "1" located outside the screen using special control buttons. By clicking on a tape cell, you can change its contents (in the "classic" version, "0" will be replaced by "1" or vice versa, in the triple version, characters "space", "0" and "1" will be changed cyclically).

You can save the state of tape in the internal buffer and load it back.

Settings allow you to change machine mode (binary or triple alphabet), as well as to choose the speed of execution.

First column in program table contains row numbers and it is filled automatically. In the second column you can select the desired command from the list, and in the third you can enter row number to jump to (if it's necessary).

Fourth column can contain a comment for selected row. You can add and delete table rows, but first you need to select row by clicking on it's number.

Executing row has a green background.

It is possible to run program in step-by-step mode, specifying start row.

Programs for Post machine can be saved in files. When you load a file and save it, the initial state of tape is automatically written to the buffer. To open "Save as" dialog box you need to hold button with the floppy disk icon.

Possible errors in execution of commands

  • > - impossible to move pointer if it is placed on the rightmost position;
  • < - impossible to move pointer if it is placed on the leftmost position;
  • X - cannot erase the label if it has already been erased;
  • 0 - cannot write to cell "0" if it already contains "0";
  • 1 - cannot write to cell "1" if it already contains "1".

Also, for the above commands and command ? you cannot jump to the nonexistent row.
